## Notification Module

It contains three part.

- NotificationModule: It need to import for navigation in stack-navigators (modular structure not StackNavigator)

- NotificationModal: It is the modal which will be shown on screen when notification come. Inside this firebase messaging called so keep inside NavigationContainer of StackNavigator file. Also this will handle notification permission.

- NotificationBell: This is the bell icon with api that will show uread notification count.
